# Playwright E2E

This workspace runs end-to-end tests against the Todo app with Datadog-friendly correlation headers.

## Correlation headers

Every request includes:
- `x-e2e-run-id`
- `x-e2e-source=playwright-e2e`
- `x-e2e-test-name`

Use these in Datadog log and trace queries to isolate one test run.

## Install

From `app/`:

```bash
npm install
npm run test:e2e:install
```

## Run Local

Make sure client/server/db are up, then run:

```bash
npm run test:e2e
```

Optional:
- `LOCAL_BASE_URL` (default `http://localhost:5173`)
- `E2E_RUN_ID` (default autogenerated)

## Run EC2

```bash
EC2_BASE_URL=http://<ec2-public-ip>:3001 npm run test:e2e:ec2
```

## Run Both Local + EC2

```bash
EC2_BASE_URL=http://<ec2-public-ip>:3001 npm run test:e2e:all
```

Optional:
- `E2E_RUN_ID`

## Full suite behavior

- UI flow: create, toggle, delete todo
- API flow: update title + negative validations
